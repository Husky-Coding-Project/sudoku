Here are some tips and tricks to up your Sudoku game!

Try looking at which values can possibly go into a square; if there is only one possible value, then the square must hold that value. You can extend this and look at an entire row, column or box using this strategy!

No guessing is required to solve a sudoku puzzle; think carefully about how a value in one square would affect the entire puzzle!

You can find more advanced strategies here if you'd like to up your game even further!
